# World-Culture-Landscape

一个简约舒适的世界人文风情静态展示站，支持中英双语 + 交互地图 + 多条件筛选。  
默认部署在 GitHub Pages，零后端、零配置。

## 一键上线步骤
1. 点击右上角 「Use this template」 → 创建你的仓库。
2. 进入仓库 Settings → Pages → Source 选择 「Deploy from a branch」 → Branch 选 `main` / (root) → Save。
3. 稍等 30 秒，访问 `https://你的用户名.github.io/仓库名/` 即可看到 live 站点。
